Place optional manual Doom textures in this folder.

Supported filenames:
- `wall-front.png` / `.jpg` / `.jpeg` / `.webp` / `.gif` / `.svg`
- `wall-side.png` / `.jpg` / `.jpeg` / `.webp` / `.gif` / `.svg`
- `ceiling.png` / `.jpg` / `.jpeg` / `.webp` / `.gif` / `.svg`
- `floor.png` / `.jpg` / `.jpeg` / `.webp` / `.gif` / `.svg`

The renderer uses flat fallback colors when a texture is missing.
